HaveYouSeen is an Android App which allows you to display the latest movie releases in a list and open a detailed view on click.
Additionally, it offers a searchbar to filter the list for their titles.

Build this project directly with gradle.

Additional dependencies:

Default android libraries:
    'com.android.support:appcompat-v7:26.+'
    'com.android.support:support-v4:26.+'
    'com.android.support:recyclerview-v7:26.+'
    'com.android.support:design:26.+'
    'com.android.support.constraint:constraint-layout:1.0.2'
    
Volley for async tasks:
    'com.android.volley:volley:1.0.0'
